# IROK RA68 全铝磁轴键盘系列规格参数

RA68 系列是艾石头品牌的旗舰之作，采用 68% 布局，集顶奢工艺与巅峰性能于一身。全系采用 **CNC 全铝合金机身**，并配备了 **FR4 金边定位板** 与专属**飘带设计**。内部拥有极其考究的六层填充系统，是追求极致手感与竞技性能玩家的终极选择。

---

## 1. RA68 (旗舰竞技版)
*搭载星闪 480 方案，RT 精度达到职业级 0.001mm*

### ⚡ 核心性能规格
- **方案架构：** **星闪 480 方案**
- **RT 精度：** **0.001mm**
- **响应延迟：** **0.08ms**
- **回报率：** **8000Hz (原生 8K)**
- **扫描率：** 32Khz 全键扫描 / 256Khz 单键扫描
- **轴体类型：** **清辉磁轴**

### 🏗️ 硬件设计与材质
- **外壳材质：** **CNC 全铝合金** (枪灰色/白色)
- **定位板：** **FR4 金边定位板** (极佳韧性与共振控制)
- **内部填充：** **六层豪华填充** (静音棉 + 轴下垫 + 声优垫 + EPDM 消音棉 + 空格键硅胶条 + Gasket 硅胶柱)
- **键帽材质：** 
  - **枪灰色：** 雾透侧刻键帽
  - **白色：** 热升华键帽
- **配件特色：** 附赠专属配色飘带 (枪灰色/白色)

### ⚙️ 软件与功能
- **驱动支持：** 网页驱动 / HUB 驱动
- **核心功能：** SOCD、音乐律动、TGL、MT、按键宏、MPT、END、DKS

## 2. RA68 双擎 (巅峰性能版)
*搭载星旋方案，双 MCU 全链路直连，挑战行业延迟极限*

### ⚡ 核心性能规格
- **方案架构：** **星旋方案 (StarSpin)**，**双 MCU 全链路直连**
- **RT 精度：** **全键全段 0.001mm (极致一致性)**
- **响应延迟：** **0.07ms (极致巅峰响应)**
- **回报率：** **8000Hz (原生 8K)**
- **扫描率：** 512K 全键扫描 / 147500K 单键扫描
- **轴体类型：** **清辉磁轴 PRO**

### 🏗️ 硬件设计与材质
- **外壳材质：** **CNC 全铝合金** (枪灰色/白色)
- **定位板：** **FR4 金边定位板**
- **内部填充：** 六层豪华填充系统 (配置同 RA68)
- **键帽材质：** **碳晶键帽** (触感更细腻，耐磨性更高)
- **连接模式：** 有线连接

### ⚙️ 软件与功能
- **核心特色：** 支持 1680 万色 ARGB 灯效与音乐律动
- **竞技功能：** **SOCD (急停)**、DKS、MPT、END、TGL、MT、按键宏
- **驱动支持：** 网页驱动 / HUB 驱动

## 🚀 RA68 系列核心卖点

- **星旋双 MCU 技术 (仅限双擎版)：** 采用行业领先的双芯片架构，实现数据传输的“全链路直连”，将延迟压缩至惊人的 0.07ms。
- **顶奢 FR4 金边定位板：** 相比普通金属定位板，FR4 材质配合金边工艺，提供了更具弹性的触底手感和更优雅的视觉表现。
- **碳晶键帽：** 双擎版标配的碳晶键帽提供了极佳的指尖摩擦力与耐用度，即使长久竞技也能保持清爽。
- **重型 CNC 铝壳：** 高精度 CNC 工艺切削出的全铝机身，配合 Gasket 结构，让键盘整体重量感十足，有效吸收竞技时的按压共振。
